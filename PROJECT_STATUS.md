```
└── public/                   # 静态资源
```

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 配置环境变量
创建 `.env.local` 文件：
```env
NEXT_PUBLIC_GEMINI_API_KEY=your_gemini_api_key_here
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

### 运行开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
npm start
```
